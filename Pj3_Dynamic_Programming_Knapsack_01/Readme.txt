Para el funcionamiento de códigos

Inputs:

  Al ejecutar los códigos ya compilados, pedirán un archivo con extensión, se debe especificar la ruta. Por ejemplo: input.txt

  Estos archivos de texto deben tener el siguiente formato para su correcto funcionamiento, a continuación se detalla el formato.

- Para 01knap.cpp y valKnap.cpp: <n> <Capacidad> <Lista pi,vi>.

  Ejemplo: 4 8 1 2 3 4 5 6 7 8

- Para aproxKnap.cpp: <n> <Capacidad> <beta> <Lista pi,vi>

  0 < beta < 1

  Ejemplo: 4 8 0.5 1 2 3 4 5 6 7 8


Outputs:

  Muestra por consola el siguiente formato:

  <n> <Capacidad> <Lista xi>

  Donde xi son 0 o 1. 0 si no toma el objeto "i", 1 si lo toma.
